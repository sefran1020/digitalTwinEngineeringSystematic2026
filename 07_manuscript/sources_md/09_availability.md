# Data and Code Availability

All materials supporting this review are openly available: the review protocol and
extraction schemas, the database search exports, the full screening trace
(per-record agent decisions, confidence scores, governing criteria, and final
status), the extraction dataset with verbatim supporting quotations, the
evidence-quality scoring, and the complete, end-to-end analysis and
figure-generation code — including the theme co-occurrence, multiple correspondence
analysis, and attribute-association-network pipelines. These are released in a
public GitHub repository (https://github.com/sefran1020/digitalTwinEngineeringSystematic2026)
and archived on Zenodo with a citable DOI (https://doi.org/10.5281/zenodo.20482762,
which always resolves to the latest version), under an MIT license for code and a
CC BY 4.0 license for data and text. The review followed a pre-specified protocol
(Review Protocol v2); the protocol will be registered (OSF) before any update.
